◆ getStockData()

void StockChart::getStockData ( const QString &  stockID)

getStockData Sends/issues a network request to fetch the data for a particular financial instrument / security / stock from the internet https://www.alphavantage.co/. The data fetched is a time series https://en.wikipedia.org/wiki/Time_series. This function can be called from a QML file since it is marked as Q_INVOKABLE.

Parameters
stockIDString with the Ticker Symbol or ID assigned to a particular security / stock that is to be fetched from the internet
